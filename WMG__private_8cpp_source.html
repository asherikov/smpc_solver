234  // z coordinates for these two points are derived as follows:
235  //
236  // S = sum of weighted binomial coefficients
237  // 0.5^3 * w0*z0 + 3*0.5^3 * w1*z1 + 3*0.5^3 * w2*z2 + 0.5^3 * w3*z3 = step_height * S
